Sect Machina

Made by a team of 15 students at Chapman University using Unity, Sect Machina is the first game I worked on that has been published. I was Lead Programmer for this game as well as the Version Control Manager of our Unity Version Control repository. It went on to win the "Most Polished Game" Award at the IEEE GameSIG Showcae 2023. Gained experience modifying pre-existing codebases and adding new functionality in order to create a more unique game experience. Learned how to work with different roles (i.e. artists, producers, music) to include new features which allowed for implementation of these other focuses.
